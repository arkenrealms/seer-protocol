# arken/packages/seer/packages/protocol/src

Seer protocol source root.

## Key files
- `router.ts`: top-level tRPC router composition (core Arken node routers + Seer domain modules).
- `types.ts`: application/router context and service/model type wiring.
- `index.ts`: package exports.
- `modules/`: Seer domain module routers/services/schemas.
